# Credit Risk and Bankruptcy Prediction System

## Overview

This project combines SQL, Power BI, and Machine Learning to analyze banking customer data and identify customers who may be at risk of financial distress.

The project follows an end-to-end analytics workflow, starting from raw banking datasets, performing SQL-based data cleaning and analysis, creating interactive Power BI dashboards, and applying machine learning techniques to predict potential bankruptcy risk.

## Tools and Technologies

* MySQL
* SQL
* Power BI
* Python
* Pandas
* Scikit-Learn
* GitHub

## Dataset

The project uses four datasets:

* Customers
* Accounts
* Transactions
* Loans

The datasets contain customer demographics, account information, transaction history, loan details, income information, and repayment behavior.

## Machine Learning Model

A Random Forest Classifier was implemented to predict customer bankruptcy risk using financial and behavioral attributes such as:

* Annual Income
* Monthly Income
* Outstanding Debt
* Monthly EMI
* Missed Payments
* Account Balance
* Employment Status
